## نصب skill سقراطی (socratic-method)

### تصمیمات کاربر
- **نسخه:** `socratic-method` از [malkreide/socratic-method-skill](https://github.com/malkreide/socratic-method-skill) (نسخه‌ی کامل با ۴ مود کارکرد)
- **محل نصب:** User-level → `C:\Users\hp\.agents\skills\`

### بررسی انجام‌شده
ساختار skillهای موجود رو بررسی کردم:
- هر skill یک پوشه‌ست در `C:\Users\hp\.agents\skills\` که داخلش `SKILL.md` قرار داره.
- فرمت فرانت‌متر: فقط `name` و `description` (مثل `code-reviewer` و `team-doc-generator`).
- ممکنه فایل‌های کمکی داشته باشه (مثل `references/` و `assets/`) ولی برای این skill فقط `SKILL.md` لازمه.
- فعلاً workspace-level skillای در `H:\bank-statement-analyzer` وجود نداره.

skill سقراطی هم دقیقاً همین فرمت رو داره (`name: socratic-method` + `description`)، پس کاملاً سازگاره و نیازی به تغییر فرمت نیست.

### مراحل پیاده‌سازی

1. **ساخت پوشه‌ی skill**
   `C:\Users\hp\.agents\skills\socratic-method\`

2. **دریافت SKILL.md مستقیم از GitHub (raw)**
   فایل raw رو مستقیماً از `https://raw.githubusercontent.com/malkreide/socratic-method-skill/main/SKILL.md` می‌گیرم و داخل پوشه ذخیره می‌کنم. این کار رو به‌جای دستی نوشتن انجام می‌دم تا فرمت markdown دقیق و بدون خطای نسخه‌برداری بمونه (escapeهای دستی رو حذف می‌کنه، backtickها و asteriskها رو درست حفظ می‌کنه).

3. **بررسی فرانت‌متر**
   مطمئن میشم که فرانت‌متر با `---` شروع/تمام میشه و `name` و `description` رو داره، و فایل قابل خواندنه.

4. **تأیید نصب**
   فایل رو نشون می‌دم تا کاربر ببینه نصب شده. بعد از restart کردن کلاینت ZCode، skill توی همه‌ی پروژه‌ها در دسترس خواهد بود.

### محتوای skill (خلاصه)
این skill کلود/مدل رو به یک معلم سقراطی تبدیل می‌کنه — به‌جای پاسخ مستقیم، با پرسشگری سیستماتیک کاربر رو به کشف دانش هدایت می‌کنه. شامل:
- **۴ مود کارکرد:** آموزشی (پیش‌فرض)، درمانی (CBT)، تحلیلی/حقوقی، استراتژیک/کوچینگ
- **ماشین حالت سقراطی:** ۵ حالت (اکتشاف اولیه → آزمون فرضیه → Aporia/گرفتگی → تعمیق → ترکیب و جمع‌بندی)
- **۶ نوع سؤال:** شفاف‌سازی، کشف مفروضات، شواهد، دیدگاه‌ها، پیامدها، متا-سؤالات
- **safeguards:** مدیریت تنش روانی، مرزهای اخلاقی، تعادل سوگیری
- **سفارشی‌سازی دامنه‌ای:** ریاضی/STEM، درمان، استراتژی کسب‌وکار، تحلیل متن

### نکته
محتوای skill انگلیسیه و به همان شکل اصلی نگهش می‌دارم تا وفادار به منبع بمونه. (اگه بخوای می‌تونم بعداً فقط بخش `description` رو به فارسی ترجمه کنم تا بهتر trigger بشه، ولی بدنه رو انگلیسی نگه می‌دارم.)
